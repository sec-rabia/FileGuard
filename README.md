# FileGuard

## Objective
FileGuard is a cybersecurity tool developed in Python to verify the integrity of files using the SHA-256 hashing algorithm.

## Features
- Generate SHA-256 hash
- Save hash
- Verify file integrity
- Detect file modifications

## Technologies Used
- Python
- hashlib

## Files
save_hash.py
verify_hash.py
hashes.txt

## How to Run

Run:

python save_hash.py

Then:

python verify_hash.py
